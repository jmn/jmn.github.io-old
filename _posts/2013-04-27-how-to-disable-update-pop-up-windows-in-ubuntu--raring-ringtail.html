  1. Run dconf-editor.

There are ways of keeping your system up to date that do not include intrusive pop up windows.

Update: Well, that did not work after all. I still got notifications, and later I found two processes consistently using a lot of CPU: apt-check and oneconf service.

Solution: 

apt-get remove update-notifier oneconf update-notifier-common

For some reason, removing this last one (update-notifier-common) also wants to remove flashplayer and MS ttf-mscorefonts-installer)

I asked on IRC (#ubuntu) about this and it turns out that flashplayer and mscorefonts are using the the update-notifier-common package to because for downloading files outside of apt. Or something like that.

These are the other packages that depend on update-notifier-common: 

apt-cache rdepends update-notifier-common

Reverse Depends:
  muon-notifier
  ttf-mscorefonts-installer
  flashplugin-installer
  zentyal-core
  update-notifier-kde
  muon-notifier
  update-notifier
  byobu

So I will install these packages another way if I need them.
