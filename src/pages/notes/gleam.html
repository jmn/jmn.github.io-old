1.1 Meta

This document should be accessible at https://jmnorlund.net/notes/gleam

1.2 Introduction

Gleam is a language which runs on Erlang/OTP, it is is statically typed and compiled erlang.

1.3 Example code

import gleam/string

pub fn hello_world() -> String {
  "Hello, from gthree!"
}

pub fn hi(hi: String) -> String{
  string.concat(["hi, ", hi])
}

1.3.1 Type checking example:

pub fn moo() -> String {
  hi(1)
}
